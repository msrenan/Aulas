# Entrada e saída
A "troca de mensagens" que ocorre para que a entrada e saída ocorra entre diferentes dispositivos é feita através de um **arquivo**. Essa troca de mensagens pode ocorrer *via bloco* ou *via caracter*.

Se a troca de mensagens ocorre via blocos, existe interferência/mediação do sistema de arquivos para garantir a boa manipulação dos blocos de memória.
